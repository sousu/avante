# avante

Control programs for [TAMIYA RC AVANTE](https://www.google.com/search?q=TAMIYA+RC+AVANTE) with Raspberry pi 4.

## function

Server

 - Motor & Servo PWM control

Client

 - On board streaming
 - Web joysticks and Gamepad API
